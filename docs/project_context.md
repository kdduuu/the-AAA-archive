# Project Context — The AAA Archive

## Sobre o Projeto

O **The AAA Archive** é um projeto pessoal desenvolvido para estudar Análise de Dados, Engenharia de Software e Desenvolvimento Full Stack através da construção de um arquivo digital dedicado à preservação da história dos videogames.

O projeto começou como uma forma de aprender Python e Pandas utilizando um dataset próprio, mas evoluiu para a construção de um software completo, composto por um backend modular, banco de dados, API e website.

Seu desenvolvimento é incremental, permitindo que cada etapa contribua para a evolução do sistema e para o aprendizado de novas tecnologias.

---

# Objetivos

O projeto possui objetivos técnicos e educacionais.

Durante seu desenvolvimento, busca-se:

* Aprender Python aplicado à Análise de Dados.
* Aprender Pandas através de um projeto real.
* Aprender PostgreSQL para persistência dos dados.
* Desenvolver uma API para disponibilização das informações.
* Construir um website moderno para exploração da coleção.
* Criar dashboards para análise dos dados.
* Aplicar boas práticas de Engenharia de Software.
* Desenvolver um projeto sólido para portfólio.

---

# Filosofia Editorial

O The AAA Archive não pretende catalogar todos os jogos já produzidos.

Seu objetivo é preservar jogos considerados relevantes para a evolução da indústria dos videogames através de uma coleção cuidadosamente curada.

Essa coleção recebe o nome de **Foundation Collection** e representa o núcleo permanente do projeto.

No futuro, novos títulos poderão ser incorporados através das **Expansion Collections**, preservando sempre a identidade editorial do Archive.

O histórico do **Game Awards** faz parte do projeto, porém é tratado como uma base de dados independente da Foundation Collection.

# Diretrizes

Durante todo o desenvolvimento serão seguidos os seguintes princípios:

* Organização antes da implementação.
* Documentação antes do código.
* Evolução incremental.
* Código limpo e reutilizável.
* Estrutura modular e escalável.
* Cada módulo deve possuir apenas uma responsabilidade.
* Toda funcionalidade deve possuir uma aplicação real dentro do produto.

# Objetivo Final

Construir um sistema completo capaz de preservar, organizar, analisar e disponibilizar informações sobre videogames através de um backend modular, uma API, um website moderno e um dashboard analítico.

Mais do que um catálogo de jogos, o The AAA Archive busca se tornar um arquivo digital organizado, consistente e preparado para evoluir continuamente.
