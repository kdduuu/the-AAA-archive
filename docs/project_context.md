# Project Context - The AAA Archive

## Sobre o Projeto

The AAA Archive é um projeto pessoal desenvolvido para estudar Análise de Dados, Engenharia de Software e Desenvolvimento Full Stack através da construção de um arquivo histórico de jogos AAA Single Player.

O projeto evoluirá gradualmente desde um simples dataset em CSV até uma aplicação completa utilizando PostgreSQL e Streamlit.

---

# Objetivos

- Aprender Pandas
- Aprender PostgreSQL
- Construir dashboards com Streamlit
- Desenvolver boas práticas de Engenharia de Software
- Criar um projeto forte para portfólio

---

# Filosofia Editorial

O projeto não pretende catalogar todos os jogos já produzidos.

Seu objetivo é preservar jogos AAA Single Player considerados relevantes para a evolução da indústria.

A Foundation Collection representa a coleção permanente do projeto.

Novos títulos serão adicionados através das futuras Expansion Collections.
